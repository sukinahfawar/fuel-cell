How to Use:

  1. Drag components from the parts area and drop them into any zone in the fuel cell
  2. You can place any component in any zone, but the fuel cell will only work when all components are in their correct positions
  3. When all components are correctly placed, electric current will flow and the bulb will light up
  4. You can reset the simulation at any time using the "Reset Simulation" button
